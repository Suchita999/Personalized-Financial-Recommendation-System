## How to Use (Step by Step)

### 1. Open in VS Code

Open the `financial_advisor/` folder in VS Code:
- File → Open Folder → select `financial_advisor`

### 2. Install Dependencies

Open VS Code terminal (Ctrl+` or View → Terminal):

```bash
pip install -r requirements.txt
```

### 3. Run the ML Pipeline (Scripts 01 → 02 → 03)

These train your models and save everything to `data/processed/`.

```bash
# Make sure you have intrvw24.zip in this folder for CE Survey data

python 01_Feature_Engineering_and_KMeans.py
# → Loads UCI Census, CE Survey, Kaggle Funds
# → Feature engineering on all 3 datasets
# → K-Means clustering on UCI (K=10)
# → Saves: census_clustered.csv, fund_features.csv, kmeans_model.pkl, etc.

python 02_Supervised_Classification.py
# → K-Means on CE Survey (K=2) → spending personas
# → Trains LR, RF, XGBoost to predict risk_tolerance
# → Saves: best_classifier.pkl, ce_kmeans_model.pkl, etc.

python 03_Fund_Matching_and_RAG.py
# → Loads all models from scripts 01 & 02
# → Builds 6 composite preference vectors
# → Cosine similarity matching → top 10 funds per profile
# → Saves: fund_recommendations_all_profiles.csv
```

### 4. Run the Web App

```bash
# Optional: set free Gemini API key for chatbot
export GOOGLE_API_KEY="AIza-your-key"    # get free at https://ai.google.dev

# Launch the app
streamlit run streamlit_app.py
```

Open **http://localhost:8501** in your browser.

### 5. Deploy to Streamlit Cloud (Free)

```bash
# Push to GitHub
git init
git add .
git commit -m "Financial advisor capstone"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/financial-advisor.git
git push -u origin main
```

Then:
1. Go to **https://share.streamlit.io** → Sign in with GitHub
2. **New app** → select repo → branch `main` → file `streamlit_app.py`
3. Click **Deploy**
4. Settings → Secrets → add `GOOGLE_API_KEY = "AIza-your-key"`
